# Blockquote

Affiche une citation et son auteur.

## Frontmatter
```yml
blocks:
  blockquote:
    text:
    author:
      title:
      text:
      image:
        src:
        alt:
```

## Netlify CMS
- Blockquote `object`
  - Texte `string`
  - Auteur `object`
    - Titre `string`
    - Texte `string`
    - Image `object`
      - Source `image`
      - Alt `string`
