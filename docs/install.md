- [Cloner le repository](#cloner-le-repository)

## Cloner le repository

Avec ses submodules
```
git clone git@github.com:sebousan/hugolify-template.git --recurse-submodules
```

## Installer Hugo

Avec brew
```
brew install hugo
```

## Installer les packages

Avec yarn
```
yarn
```
