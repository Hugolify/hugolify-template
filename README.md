## Features
Hugo sections available in theme and admin

* pages
* posts
  * authors (taxonomy)
  * categories (taxonomy)
  * tags (taxonomy)
* projects
  * types (taxonomy)
  * tags (taxonomy)
* casestudies
* publications
  * concerned (taxonomy)
